# **App Name**: MarketWatch AI

## Core Features:

- Live Stock Market Tracker: Displays real-time stock prices from APIs like Alpha Vantage or Finnhub.
- Sentiment Analysis: Analyzes news headlines and articles for sentiment (positive, negative, neutral).
- Watchlist Personalization: Save custom stock watchlists.  Real-time updates for selected stocks and news articles.
- Custom Alerts: User-configurable alerts for price changes.  In-app notifications.
- Analytics Dashboard: Displays top gainers, losers, and most active stocks.  Visual insights.
- AI Investment Assistant: AI-powered tool that responds to queries with current insights.

## Style Guidelines:

- Primary color: A deep blue (#3F51B5) to evoke trust and stability, aligning with the financial context.
- Background color: A very light gray (#F0F2F5), nearly white, providing a clean and professional backdrop.
- Accent color: A vibrant purple (#7E57C2), used sparingly to highlight key interactive elements, drawing the user's attention.
- Clean and professional font for readability.
- Minimalist and modern icons for quick visual understanding.
- A clean, card-based layout.
